# AI 提示词：创建现代化个人网站

请创建一个类似 Albert 个人网站风格的现代化、响应式个人网站。以下是详细要求：

## 网站核心功能
- 响应式导航栏（桌面和移动端适配）
- 英雄区域带个人头像和简介
- 关于我区域包含时间线和兴趣标签
- 项目展示区域带分类过滤功能
- 社交媒体链接卡片
- 技能图标展示区域
- 联系我区域带模态框弹出二维码
- 明暗主题切换功能（白天/黑夜模式）
- 固定背景图片效果

## 设计风格要求
- 使用优雅的蓝色主题配色方案
- 现代简约设计，注重用户体验
- 响应式布局，适配各种屏幕尺寸
- 平滑的过渡动画效果
- 毛玻璃效果（frosted glass）的容器背景

## 技术实现要求
- HTML、CSS、JavaScript 完全分离
- 使用 CSS 变量实现主题切换
- 模块化的 CSS 结构（主样式表 + 组件样式）
- 使用 Google Fonts（Poppins 和 Noto Sans SC）
- 使用 Font Awesome 图标
- 背景图片从 Unsplash 获取高质量图片

## 具体内容结构

### HTML 结构
创建语义化的 HTML5 结构，包含：
- 固定导航栏带 logo 和菜单项
- 英雄区域（姓名、职业、简介、按钮）
- 关于我区域（个人介绍、时间线、标签）
- 项目展示区域（带过滤功能的卡片网格）
- 社交媒体链接区域（B站、微博等卡片）
- 技能展示区域（图标+文字网格）
- 联系我区域（邮件、微信、QQ按钮）
- 页脚
- 模态框（微信和QQ二维码）

### CSS 样式
实现：
- CSS 变量定义明暗主题颜色
- 固定背景图片效果（从 Unsplash 获取）
- 响应式网格布局
- 毛玻璃效果的容器背景
- 按钮和链接的悬停效果
- 移动端适配的导航菜单
- 项目卡片和社交媒体卡片的样式
- 模态框动画效果

### JavaScript 功能
实现：
- 移动端导航菜单切换
- 滚动时导航链接高亮
- 项目分类过滤功能
- 模态框的打开和关闭
- 明暗主题切换和本地存储
- 平滑滚动效果

## 视觉设计指导
- 主色调：优雅蓝色 (#007bff, #0056b3, #56aeff)
- 字体：Poppins（英文）和 Noto Sans SC（中文）
- 背景图片：从 Unsplash 获取高质量风景或抽象背景图
- 图标：使用 Font Awesome 6
- 布局：使用 CSS Grid 和 Flexbox
- 间距：合理的留白和间距

## 响应式要求
- 桌面端：完整导航菜单
- 平板端：适配中等屏幕
- 移动端：汉堡菜单，单列布局

## 额外要求
- 代码注释清晰
- 文件结构组织良好
- 性能优化（图片压缩等）
- 浏览器兼容性考虑
